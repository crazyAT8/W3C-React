React Render HTML

React's goal is to render HTML in a web page

React renders HTML to the web page by using a function called createRoot() and its method render()

The createRoot Function

  • the createRoot() takes one argument, an HTML element.
  • the purpose of the function is to define the HTML element where a React component should be displayed.

The render Method

  • the render() method is then called to define the React component that should be rendered
  • which is a index.html file in the "public" folder in your React project

The HTML Code

  • JSX which allows you to write html tags inside the JavaScript code
  • basically create a const variableName = (html code surrounded by a <div> tag)
  • the html code within the circle brackets is merely a segment or section of the HTML
  • you write out that code and ship it to your index with this code
    import React from 'react';
    import ReactDOM from 'react-dom/client';

    const variableName = (
       // HTML code
    )

    const container = document.getElementById('root');
    const root = ReactDOM.createRoot(container);
    root.render(variableName)
